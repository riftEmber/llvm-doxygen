Detailed Description

Macro Info DWARF-like metadata node.

A metadata node with a DWARF macro info (i.e., a constant named DW_MACINFO_*, defined in llvm/BinaryFormat/Dwarf.h). Called DIMacroNode because it's potentially used for non-DWARF output.
