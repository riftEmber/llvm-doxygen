Detailed Description

template<class ELFT>
struct llvm::object::Elf_Verdef_Impl< ELFT >

Elf_Verdef: This is the structure of entries in the SHT_GNU_verdef section (.gnu.version_d).

This structure is identical for ELF32 and ELF64.
