Detailed Description

This class keeps offsets to the debug sections.

Any object which is supposed to be emitted into the debug section should use this class to track debug sections offsets.

uint64_t llvm::dwarflinker_parallel::OutputSections::getStartOffset ( DebugSectionKind SectionKind) const
inline

When objects(f.e.

compile units) are glued into the single file, the debug sections corresponding to the concrete object are assigned with offsets inside the whole file. This method returns offset to the SectionKind debug section, corresponding to this object.
