45/// Create a LinkGraph from a MachO/arm64 relocatable object.

53/// jit-link the given object buffer, which must be a MachO arm64 object file.
54///
55/// If PrePrunePasses is empty then a default mark-live pass will be inserted
56/// that will mark all exported atoms live. If PrePrunePasses is not empty, the
57/// caller is responsible for including a pass to mark atoms as live.
58///
59/// If PostPrunePasses is empty then a default GOT-and-stubs insertion pass will
60/// be inserted. If PostPrunePasses is not empty then the caller is responsible
61/// for including a pass to insert GOT and stub edges.
62void link_MachO_arm64(std::unique_ptr<LinkGraph> G,
63 std::unique_ptr<JITLinkContext> Ctx);
