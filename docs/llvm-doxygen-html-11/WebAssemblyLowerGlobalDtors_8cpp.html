Detailed Description

Lower @llvm.global_dtors.

WebAssembly doesn't have a builtin way to invoke static destructors. Implement @llvm.global_dtors by creating wrapper functions that are registered in @llvm.global_ctors and which contain a call to __cxa_atexit to register their destructor functions.
